# Next.js Starter Template

An opinionated Next.js (App Router) starter with TypeScript, strict linting, and a styled landing page to begin from.

## Getting started

1. Install dependencies: `npm install`
2. Run the dev server: `npm run dev` then open `http://localhost:3000`
3. Lint the project: `npm run lint`
4. Build for production: `npm run build` then `npm run start`

## Features

- App Router layout with metadata and global styles
- Strict TypeScript config with typed routes enabled
- ESLint using `next/core-web-vitals`
- Ready-to-deploy scripts for development, build, and production start

## Notes

- Requires Node.js 18.17+ or 20+
- Update `package.json` versions as needed to match your toolchain
